Merge Requirements
==================

XDG Desktop Portal pull requests have a number of requirements that need to be
fulfilled, before it can get merged. The requirements are different, depending
on what part of the project is being changed.

- Any change requires at least one review by one of the committers
- Changes to the Portal frontend or backend API require an ACK from a
  representative of every listed desktop environment
- Changes to the Flatpak or Snap XdpAppInfo subclasses, and changes to the
  XdpAppInfo class which affect the Flatpak or Snap subclasses require an ACK
  from a representative of the respective project

Those rules should be understood as strong guidelines. If in doubt, we should
try to talk with each other on Matrix or via video chat if necessary.
